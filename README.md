# calllib

## Install

```INSTALLATION
npm install --save calllib bootstrap reactstrap
```

## Usage

```js
import React, { Component } from "react";

import CallHistory from "calllib";

class Example extends Component {
  render() {
    return <CallHistory />;
  }
}
```
